# 🌦️ Weather App

A simple and responsive Weather App that shows real-time weather data using an API.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Shows temperature, humidity, and weather condition
* 📱 Responsive design (mobile + desktop)
* ⚡ Fast and user-friendly UI


##  Technologies Used

* HTML
* CSS
* JavaScript
* Weather API (OpenWeather / any API you used)
